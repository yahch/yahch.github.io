在树莓派B上安装 TP-LINK TL-WN823N 无线网卡

2019/3/10 15:24:46

Source:https://www.raspberrypi.org/forums/viewtopic.php?t=198015

The TL-WN823N v2 you have uses the 8192eu driver. The driver in not included in the Raspbian image and needs installing.

I have a script that will install a driver I compile if you are using the RASPBIAN OS.

You can download and install the script using commands

sudo wget http://fars-robotics.net/install-wifi -O /usr/bin/install-wifi
sudo chmod +x /usr/bin/install-wifi

With the wifi dongle connected to the Pi to install the driver run command

sudo install-wifi

Changes (wlan0 can have another name if you're using Raspberry PI 3):

  1. Instead of using /etc/network/interfaces, I've added the definition block to /etc/dhcpcd.conf
interface wlan0
static ip_address=192.168.42.1/24
static ip6_address=fd0d:e8cf:c363:2605::1/64
static routers=192.168.42.1

  1. Comment out driver line in the proposed /etc/hostapd/hostapd.conf version

  2. No need for Update hostapd (maybe) section (kernel v4.14 at the time I'm writing this).

  3. Did not need to remove wpa_supplicant. Beware of this if you want to keep using your Raspi 3 as a wifi client (I assume this is what you wanted buying an extra wifi device).

  4. I used systemctl instead of service command
